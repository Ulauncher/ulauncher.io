Instant app launcher

Ulauncher provides instant search results. It also remembers your previous choices and automatically selects the best option for you.

Fuzzy search

​ Type in an application name without worrying about spelling. Ulauncher will figure out what you meant.

Fast directory browser

​ Use Ulauncher to open your files and directories faster with fuzzy search.

Install Ulauncher

Tested only on latest stable releases of Debian-based Linux distributions.

Install from PPA: 
sudo add-apt-repository ppa:agornostal/ulauncher
sudo apt-get update
sudo apt-get install ulauncher
We are working on creating RPM build. Any help with this is highly appreciated! See the Github issue #27.
See how to install from sources on your Linux machine.

Roadmap

v1.0 (Current)

  • Search among installed applications
  • Default search options: Google and Wiki
  • File and directory browser

v2.x (2016 Q1)

  • Support rpm-based distributions
  • Redesigned settings
  • Search documents and files

v3.0 (2016 Q2)

  • Workflows.
    Users will be able to create their own workflows. For example, create new result items for Ulauncher that when triggered will run custom system commands, or show user defined result items, or show system notifications, etc.
    Programming skills will not be necessary.

v4.0 (2016 Q3)

  • Plugin API.
    Developers will be able to create Ulauncher plugins that run along with the app and may extend Setting.
  • Plugin website.
    Developers will be able to publish their plugins.
    Users will be able to search and install plugins.
    User ratings and comments will be available.
