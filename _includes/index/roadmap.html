• Extenson API. Dev Release Completed

    Extension API v1 is now available in dev release.
    You can now create custom extensions with Python that improve your personal workflow.

    Start building your etensions!
    Read more at docs.ulauncher.io

  • Configurable Extension Preferences

    If you know Angular please contribute to Ulauncher. Your help is needed to implement UI for managing preferences.

    Once this task is complete users will be able to configure extension preferences through Preferences window.

    Check out details in issue #121

  • Add, Remove, and Update Extensions

    Next step is to add Add, Remove, and Update functionality so that users won't have to manually install and update extensions.

  • Website for Extensions

    There will be a website where users can discover new and share their own extensions.

    If you know Python or Node.js please contribute to website development.

    Contact us via an email.

  • More result views

    We want to build more result views available via Extension API so developers are not limited to the default result view (list view with icon on the left and name/description on the right).

    Eventually there will be multiple list views.
