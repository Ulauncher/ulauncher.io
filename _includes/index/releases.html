Ulauncher is available for:
  • Debian-based distros (Ubuntu, elementary OS, etc.): loading link...
    sudo dpkg -i packagename.deb or just double click
    Note: if dpkg -i exits with error about dependency problems, run sudo apt-get install -f
  • Arch Linux: Latest release is available on AUR
    git clone https://aur.archlinux.org/ulauncher.git && cd ulauncher && makepkg -is
  • Fedora: loading link...
    sudo dnf install packagename.rpm
  • OpenSUSE: loading link...
    sudo zypper in packagename.rpm or double click
